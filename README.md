# harulife
하루살이는 오늘의 뉴스, 정보, 편의기능을 한번에 보자는 모토로 제작되었습니다.

# 기능
## 콘텐츠
* 메인 뉴스 (연합뉴스)
  * 오늘의 최신 뉴스들을 확인해보세요. 매일매일 업데이트되는 새로운 기사들을 살펴볼 수 있습니다.
* 날씨 
  * 오늘 날씨, 앞으로 5일의 날씨를 확인해볼 수도 있습니다. 또한 날씨 기사도 확인해보세요. (현재 서울특별시만 지원, 추후 다양한 지역/국가을 지원할 예정입니다.)
* 라이프 스토리 (오마이뉴스)
  * 사는 이야기를 들어보세요. 여러 생활 팁들과 살아가며 느끼는 이야기 등 많은 이야기들이 있습니다.
## 편의기능
* QR체크인 바로가기 
  * QR체크인에 바로 접근할 수 있습니다. (네이버 QR체크인 바로가기)
* 오늘 할 일
  * 오늘 해야할 일들을 잊지 않게 적어둘 수 있습니다.
* 우편번호 찾기
  * 우리집의 우편번호를 편리하게 조회할 수 있습니다.

# 안내
* 사용된 언론사는 저의 어떠한 정치적 견해 및 성향과 무관합니다.
* apk 파일로 설치할때 구글 프로텍트가 차단함 문구가 나올 수 있는데, 구글 개발자 등록이 안되어서 그런 것이니 <b>무시하고 설치</b>를 눌러주세요.
## 사용 권한 (안드로이드 앱 사용시)
* 위치 
  * 지도상에서 나의 위치를 확인하기 위해서 위치 권한을 사용합니다. (위치 정보는 절대로 저장되지 않습니다.)
* 저장
  * 이미지 저장, 앱 속도 향상을 위한 캐시 저장을 위해 저장 권한을 사용합니다.
* 카메라
  * 게시물 이미지, 프로필 이미지 업로드를 위하여 카메라 권한을 사용합니다.
* 만약 권한을 거부하여도 앱 사용에 문제가 없지만, 만약 권한이 필요할 경우 따로 설정->애플리케이션->하루살이->권한에서 권한 동의를 해주셔야 합니다.
